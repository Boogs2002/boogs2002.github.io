𓁷 Origins Profile & Background

I didn't start out chasing code. My path began in construction especially my vision about architecture so i began with: — first Natin-Infrastructuur, then Natin-Bouwkunde — learning how physical structures are planned and built. But somewhere between blueprints and site plans, I found I was more curious about the systems running underneath everything else: the software. That curiosity led me to UNASAT, where I'm now in my second year studying Software Engineering.

Since then, I've been building — literally. From internal dashboards and automated reporting tools to a medical supply management system and this very site, I like projects that solve a real, specific problem for real people. Lately, that curiosity has pulled me toward the security side of development: understanding not just how to build systems, but how to break them, so I can build them stronger.
